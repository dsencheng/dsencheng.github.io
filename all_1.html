Xcode命令

命令行-单元测试

使用Cocoapods有workspace 必须加-workspace，注意使用引号及完整的文件名， [-only-testing:<test-identifier>] test-identifier格式说明 TestTarget[/TestClass[/TestMethod]]

xcodebuild test -workspace 'opensdk.xcworkspace' -scheme 'opensdk' -destination 'platform=iOS Simulator,name=iPhone 8,OS=13.5'
xcodebuild test -workspace 'opensdk.xcworkspace' -scheme 'opensdk' -destination 'platform=iOS Simulator,name=iPhone 8,OS=13.5' -only-testing:'opensdkTests'
xcodebuild test -workspace 'opensdk.xcworkspace' -scheme 'opensdk' -destination 'platform=iOS Simulator,name=iPhone 8,OS=13.5' -only-testing:'opensdkTests/LoginTests/testLoginExample'

省略了class
xcodebuild test -workspace 'opensdk.xcworkspace' -scheme 'opensdk' -destination 'platform=iOS Simulator,name=iPhone 8,OS=13.5' -only-testing:'opensdkTests/testLoginExample'

Shell 笔记

获取podspec版本号

VersionString=`grep -E 's.version.*=' xxx.podspec`
VersionString=${VersionString##*=}
VersionNumber=`tr -d "'" <<<"$VersionString"`
echo "current version is ${VersionNumber}"
